Cybersecurity Tools

Essential tools for security professionals and penetration testers

Nmap

Network Scanner

Powerful network discovery and security auditing tool. Used for network inventory, managing service upgrade schedules, and monitoring host or service uptime.

  • Port Scanning
  • OS Detection
  • Service Version Detection
  • Script Engine (NSE)
Easy Learn More

Burp Suite

Web App Testing

Integrated platform for performing security testing of web applications. Industry-standard tool for web app pen testing with powerful manual and automated features.

  • Intercepting Proxy
  • Web Vulnerability Scanner
  • Intruder (Fuzzing)
  • Repeater & Sequencer
Medium Learn More

Wireshark

Packet Analyzer

World's most popular network protocol analyzer. Lets you capture and interactively browse traffic running on a computer network in real-time.

  • Live Capture & Offline Analysis
  • Deep Protocol Inspection
  • Rich VoIP Analysis
  • Export to Multiple Formats
Medium Learn More

Metasploit Framework

Exploitation

Most widely used penetration testing framework. Provides tools for developing, testing, and executing exploit code against remote target machines.

  • Exploit Development
  • Payload Generation
  • Post-Exploitation
  • Database Integration
Advanced Learn More

Aircrack-ng

Wireless Security

Complete suite of tools to assess WiFi network security. Focuses on monitoring, attacking, testing, and cracking wireless networks.

  • Packet Capture & Export
  • WEP/WPA/WPA2 Cracking
  • Fake Access Points
  • Deauthentication Attacks
Medium Learn More

John the Ripper

Password Cracker

Fast password cracker supporting hundreds of hash and cipher types. Can automatically detect password hash types and includes a customizable cracker.

  • Dictionary Attacks
  • Brute Force
  • Rainbow Tables
  • Multi-Platform Support

Nikto

Web Scanner

Open source web server scanner which performs comprehensive tests against web servers for multiple items including dangerous files and outdated versions.

  • 6700+ Potential Vulnerabilities

OWASP ZAP

Security Testing

Zed Attack Proxy is an easy to use integrated penetration testing tool for finding vulnerabilities in web applications during development and testing.

  • Automated Spidering
  • Fuzzing
  • API Testing

Sqlmap

Automatic SQL injection and database takeover tool. Supports six SQL injection techniques and can directly connect to databases without passing through SQL injection.

  • Data Extraction

Hashcat

World's fastest password recovery tool. Supports five unique modes of attack for over 300 highly-optimized hashing algorithms using CPU, GPU, and APU.

  • Distributed Cracking
